Sobre mim

Estou matriculado na Fiap Vila Mariana, no 4° semestre do curso de Engenharia de computação (previsão de conclusão 11/2027). Essa graduação teve ênfase em desenvolvimento em python, em seus dois primeiros semestres, Java e banco de dados no terceiro e quarto. Durante as aulas lidamos com desafios colocados pela instituição alguns deles destinados a empresas de fora como por exemplo a empresa Raízen onde tive como objetivo criar uma versão melhorada do software mobile que a empresa nos apresentou(Shell box).

Além disso, estive matriculado na Escola ITB Basílio Flores de Azevedo, durante 3 anos(2020-2022), onde conclui o ensino médio técnico em informática. Esse período nesse intuição contribuiu muito para minha formação, tive meu primeiro contato com a programação, conhecendo a linguagem Java, utilizando a IDE Android Studio para realização do meu TCC, adquirindo também conhecimentos em SQL, HTML, CSS e Javascript.
